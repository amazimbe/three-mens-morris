# Three mens morris board game

This app allows two players to play three mens morris via a shared
terminal.

## Setup

Recommended Ruby version 2.7.0

Install gems.

`bundle install`

Running the application:

`./bin/three_mens_morris`

## Code linting
 
`bundle exec rubocop`

## Running Test Suite

Run all tests:

`bundle exec rspec`
